  1. Sur ton PC, lance un scraper :
     npm run scrape-complet
  
  2. Upload le nouveau resultats-cache.json :
     scp resultats-cache.json user@serveur:/var/www/html/
  
  3. Rafraîchis ton navigateur : Ctrl+F5

🐛 PROBLÈMES COURANTS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ Page blanche ?
   → Vérifie que index.html est à la racine web
   → Vérifie les permissions : chmod 644 index.html

❌ CSS ne charge pas ?
   → Vérifie que le dossier assets/ est présent
   → Vérifie les permissions : chmod -R 755 assets/

❌ Résultats ne s'affichent pas ?
   → Upload resultats-cache.json à la racine
   → Vérifie : chmod 644 resultats-cache.json

❌ Erreur 404 en rafraîchissant ?
   → Vérifie que .htaccess est présent
   → Active mod_rewrite sur Apache
