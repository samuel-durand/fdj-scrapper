┌────────────────────────────────────────────────────────────────┐
│  IDENTIFIANTS O2SWITCH                                         │
└────────────────────────────────────────────────────────────────┘

Pour FileZilla ou WinSCP :

  Hôte :     ftp.ton-domaine.com
  Port :     21 (FTP) ou 22 (SFTP)
  User :     (ton user cPanel)
  Pass :     (ton mdp cPanel)

  Dossier web : /www/ ou /public_html/


┌────────────────────────────────────────────────────────────────┐
│  CONFIGURATION WINSCP (Upload automatique - OPTIONNEL)         │
└────────────────────────────────────────────────────────────────┘

1️⃣  Installe WinSCP : https://winscp.net/

2️⃣  Édite le fichier : upload-o2switch.txt

    Ligne 13 : Change TON_USER par ton vrai user
    Ligne 13 : Change TON_PASSWORD par ton vrai mdp
    Ligne 13 : Change ftp.ton-domaine.com par ton vrai FTP
    Ligne 17 : Change /home/TON_USER/www/ par ton vrai chemin

    Exemple :
    open sftp://sam123:MonPass@ftp.monsite.com/
    cd /home/sam123/www/

3️⃣  Teste :
    Double-clique sur scripts\deployer-o2switch.bat
    Choisis W pour WinSCP
    ✅ Upload automatique !

┌────────────────────────────────────────────────────────────────┐
│  DÉPANNAGE                                                     │
└────────────────────────────────────────────────────────────────┘

❌ Les résultats ne s'affichent pas ?
   → Vérifie que resultats-cache.json est bien uploadé

❌ Le site ne charge pas ?
   → Vérifie que tous les fichiers de dist\ sont uploadés
   → Vérifie que tu es dans /www/ (pas dans un sous-dossier)

❌ WinSCP ne fonctionne pas ?
   → Vérifie tes identifiants dans upload-o2switch.txt
   → Utilise FileZilla en manuel

❌ Erreur lors du scraping ?
   → Vérifie ta connexion Internet
   → Réessaye dans 5 minutes
