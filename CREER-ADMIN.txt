📌 ÉTAPE 1 : Démarrer MongoDB
----------------------------------
Assurez-vous que MongoDB est démarré et accessible.


📌 ÉTAPE 2 : Démarrer le backend
----------------------------------
cd backend
npm run dev

(Le serveur doit être lancé sur le port 5000)


📌 ÉTAPE 3 : Créer l'utilisateur admin
----------------------------------
Dans un NOUVEAU terminal :

cd backend
node scripts/create-admin.js

Le script vous demandera :
→ Nom de l'administrateur: [Votre nom]
→ Email: [Votre email]
→ Mot de passe (min. 6 caractères): [Votre mot de passe]

⚠️ NOTEZ BIEN VOS IDENTIFIANTS !


📌 ÉTAPE 4 : Démarrer le frontend
----------------------------------
Dans un AUTRE terminal :

npm run dev

(Le site doit s'ouvrir sur http://localhost:5173)


📌 ÉTAPE 5 : Se connecter en tant qu'admin
----------------------------------
1. Ouvrir http://localhost:5173
2. Cliquer sur "Connexion"
3. Entrer votre email et mot de passe créés à l'étape 3
4. Cliquer sur "Se connecter"


📌 ÉTAPE 6 : Accéder au panel admin
----------------------------------
Une fois connecté, vous verrez :

🌙  🔔  🔐  👤 [Votre nom]

Cliquez sur le bouton 🔐 (cadenas doré) pour ouvrir le panel admin !

🆘 PROBLÈMES ?
----------------------------------
1. Le bouton 🔐 n'apparaît pas ?
   → Vérifiez dans MongoDB que votre rôle est bien "admin"
   → Déconnectez-vous et reconnectez-vous

2. Erreur "Accès refusé" ?
   → Vérifiez que le backend est bien démarré
   → Vérifiez votre token dans localStorage

3. Les statistiques ne chargent pas ?
   → Vérifiez que MongoDB est bien connecté
   → Vérifiez la console du navigateur (F12)
